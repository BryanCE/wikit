# Task 2.1: Create MCP Repository

## Purpose

Create a new separate repository for the MCP server that will import from `@bryance/wikit`.

## Steps

### 1. Create New Directory

```bash
# Navigate to your projects directory
cd ~/Documents  # or wherever you keep projects

# Create new directory
mkdir wikit-mcp
cd wikit-mcp
```

### 2. Initialize Git

```bash
git init
```

### 3. Initialize Bun Project

```bash
bun init -y
```

This creates a basic package.json.

### 4. Install Dependencies

```bash
# Install wikit package (the one you just published)
bun add @bryance/wikit

# Install MCP SDK
bun add @modelcontextprotocol/sdk

# Install dev dependencies
bun add -d typescript @types/node
```

### 5. Create Basic Structure

```bash
# Create source directory
mkdir -p src/tools

# Create TypeScript config (will update in Task 2.5)
touch tsconfig.json

# Create README (will update in Task 2.6)
touch README.md

# Create .env for configuration
touch .env

# Create .gitignore
cat > .gitignore << 'EOF'
node_modules/
dist/
.env
.DS_Store
*.log
EOF
```

### 6. Create Basic .env Template

```bash
cat > .env.example << 'EOF'
# Primary Wiki.js Instance
WIKI_PRIMARY_URL=https://your-wiki.com
WIKI_PRIMARY_API_KEY=your-api-key

# Optional Secondary Instance
WIKI_SECONDARY_URL=https://wiki2.com
WIKI_SECONDARY_API_KEY=key2
EOF
```

### 7. Verify Installation

```bash
# Check dependencies installed
bun pm ls

# Should see:
# @bryance/wikit
# @modelcontextprotocol/sdk
```

### 8. Create GitHub Repository (Optional)

```bash
# If you want to host on GitHub
gh repo create wikit-mcp --public --source=. --remote=origin
git add .
git commit -m "Initial commit"
git push -u origin main
```

Or manually create at github.com/bryance/wikit-mcp

## Directory Structure

After this task, you should have:

```
wikit-mcp/
├── .env.example
├── .gitignore
├── README.md
├── package.json
├── tsconfig.json
├── bun.lockb
├── node_modules/
│   ├── @bryance/wikit/
│   ├── @modelcontextprotocol/sdk/
│   └── ...
└── src/
    └── tools/
```
